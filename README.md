# 📊 A/B Testing – Landing Page Experiment

## 🧠 Descripción del Proyecto
 Este proyecto analiza un experimento A/B realizado sobre una landing page con el objetivo de determinar cuál versión (A o B) genera mejores resultados en términos de:

## 📈 Tasa de conversión
* 💰 Gasto promedio por usuario convertido
* 🔗 Influencia de la fuente de tráfico
* 👤 Impacto del tipo de usuario (nuevo vs recurrente)

El análisis se desarrolló en Python utilizando pruebas estadísticas para validar los resultados con rigor cuantitativo.

## 🎯 Objetivos
1. Determinar si existe diferencia significativa en la tasa de conversión entre las versiones A y B.
2. Evaluar diferencias en el gasto promedio por usuario convertido.
3. Analizar si la fuente de tráfico influye en la conversión.
4. Verificar si el tipo de usuario afecta el comportamiento de compra.

## 🛠️ Tecnologías Utilizadas
* Python
* Pandas
* NumPy
* SciPy
* Statsmodels
* Matplotlib / Seaborn
* Jupyter Notebook

## 📊 Metodología
Se aplicaron las siguientes pruebas estadísticas:
* Prueba Z de proporciones → Comparación de tasas de conversión.
* Prueba t de muestras independientes → Comparación de gasto promedio.
* Prueba Chi-cuadrado → Asociación entre variables categóricas.
Se trabajó bajo un nivel de significancia α = 0.05.

##  📈 Resultados Clave
✅ La versión B presenta una tasa de conversión significativamente mayor.

✅ La versión B genera mayor gasto promedio por usuario.

🔎 La fuente de tráfico tiene asociación estadísticamente significativa con la conversión.

❌ No se encontró relación significativa entre tipo de usuario y conversión.

## 🚀 Conclusión
La versión B supera a la versión A tanto en conversión como en rentabilidad, por lo que se recomienda su implementación como versión oficial.
